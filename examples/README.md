# rodhypix Examples

This directory contains the standalone reader example notebook.

## Notebook

### Image Visualization (`01_image_visualization.ipynb`)

Topics covered:

- Reading `.rodhypix` files with `RODImageReader`
- Displaying real-space images
- Visualizing diffraction frames
- Accessing image metadata

## Setup

```bash
pip install -e .[viz]
```

The notebook uses `example_data/exp_11317.zip`.
